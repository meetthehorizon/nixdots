# Generic Host Profile (`hosts/generic/`)

This directory contains configuration files for a standard/generic machine.

## Contents

- [default.nix](file:///home/conart/nixdots/hosts/generic/default.nix): Declares host identity `generic` and imports the generic hardware profile and shared system configs.
- [hardware-configuration.nix](file:///home/conart/nixdots/hosts/generic/hardware-configuration.nix): Automatically generated disk, device, and filesystem configuration file for the generic system.

## Usage

Mainly used as a reference configuration or for test builds using standard VMs and fallback systems.
