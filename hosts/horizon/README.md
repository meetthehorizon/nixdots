# Horizon Host Configuration (`hosts/horizon/`)

This directory contains the machine-specific configuration for the host `horizon`—the primary system (Asus Zephyrus G14).

## Configuration Files

- [default.nix](file:///home/conart/nixdots/hosts/horizon/default.nix): Declares host identity `horizon` and imports the Asus G14 hardware profiles, common system modules, and hardware configurations.
- [hardware-configuration.nix](file:///home/conart/nixdots/hosts/horizon/hardware-configuration.nix): Autogenerated filesystem, kernel module, swap partition, and boot parameters specific to the physical G14 machine.

## Rebuild Command

To rebuild and apply this specific configuration, execute:
```bash
sudo nixos-rebuild switch --flake ~/nixdots/#horizon
```
Or test it with:
```bash
sudo nixos-rebuild test --flake ~/nixdots/#horizon
```
These operations can be triggered by the user-defined aliases `nate` and `nest`.
